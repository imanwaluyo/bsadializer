Perhitungan Adekuasi Dialisis

Teori & Literatur Hemodialisis

1. Rumus BSA (Mosteller, 1987)
BSA = √((Tinggi(cm) × Berat(kg)) / 3600)

2. URR
URR = ((BUN pre − BUN post) / BUN pre) × 100
Target ≥ 65%

3. spKt/V (Daugirdas II)
spKt/V = −ln (R − 0.008t) + (4 − 3.5R)(UF/BB)

4. Target KDOQI
Minimum spKt/V ≥ 1.2
Target klinis optimal ≥ 1.4
